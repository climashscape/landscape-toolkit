# Component Reference

This directory contains detailed documentation for each Grasshopper component included in the Landscape Toolkit.

## Optimization
*   [**Wooly Path Optimizer**](WoolyPathOptimizer.md) - Bio-mimetic path optimization (Slime Mold).
*   [**Bio-Path Optimizer**](PathOptimizer.md) - Simple curve smoothing using relaxation.

## Modeling
*   [**Quad Road Network**](QuadRoadNetwork.md) - Generate high-quality quad mesh road networks.
*   [**Plot Generator**](PlotGenerator.md) - Create plot regions from closed boundaries.
*   [**Landscape Terrain**](Terrain.md) - Create smooth terrain meshes from contours or points.
*   [**Landscape Steps**](Steps.md) - Parametric step generation along paths.
*   [**Landscape Wall**](Wall.md) - Create vertical walls with thickness.

## Analysis
*   [**Slope Analysis**](SlopeAnalysis.md) - Visualize terrain slope gradients.
*   [**Solar Analysis**](SolarAnalysis.md) - Estimate solar exposure (sun/shadow map).
*   [**Hydrology Analysis**](Hydrology.md) - Simulate water runoff and drainage paths.
