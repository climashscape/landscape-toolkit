# Carbon Analysis

**Category:** Landscape > Analysis
**Nickname:** Carbon

## 1. Introduction
The **Carbon Analysis** component estimates the carbon sequestration benefits of a landscape design based on average carbon sequestration capabilities of common landscape plants.

## 2. Inputs
*   **TreePoints (T)**: List of points representing trees.
*   **GreenAreas (A)**: List of meshes/surfaces representing shrubs or lawns.
*   **TreeFactor (TF)**: Annual carbon sequestration per tree (kg/year), Default: 22.0.
*   **AreaFactor (AF)**: Annual carbon sequestration per unit area of green space (kg/m2/year), Default: 1.5.

## 3. Outputs
*   **TotalCarbon (C)**: Total carbon sequestration (kg/year).
*   **Report (R)**: Detailed calculation report (Text).

# Carbon Analysis (碳汇分析)

**分类:** Landscape > Analysis
**别名:** Carbon

## 1. 简介 (Introduction)
**Carbon Analysis** 组件用于估算景观设计方案的碳汇效益（固碳量），基于常见园林植物的平均固碳能力进行估算。

## 2. 输入 (Inputs)
*   **TreePoints (T)**: 代表乔木的点列表。
*   **GreenAreas (A)**: 代表灌木或草坪的网格/曲面列表。
*   **TreeFactor (TF)**: 单株乔木年固碳量 (kg/year)，默认 22.0。
*   **AreaFactor (AF)**: 单位面积绿地年固碳量 (kg/m2/year)，默认 1.5。

## 3. 输出 (Outputs)
*   **TotalCarbon (C)**: 总固碳量 (kg/year)。
*   **Report (R)**: 详细计算报告（文本）。
