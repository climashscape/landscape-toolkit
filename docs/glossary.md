# Glossary (术语表)

This document defines the standard terminology used throughout the Landscape Toolkit project.
本文档定义了 Landscape Toolkit 项目中使用的标准术语。

| Term (En) | Term (Zh) | Description |
| :--- | :--- | :--- |
| **Quad Mesh** | 四边面网格 | A mesh composed entirely of 4-sided faces, ideal for subdivision and smooth deformation. <br> 完全由四边形面组成的网格，适合细分和平滑变形。 |
| **SubD (Subdivision Surface)** | 细分曲面 | A method of representing a smooth surface via the specification of a coarser polygon mesh. <br> 通过粗糙多边形网格表示光滑曲面的方法。 |
| **Road Network** | 路网 | A system of interconnected roads, typically represented as a graph or mesh. <br> 相互连接的道路系统，通常表示为图或网格。 |
| **Wooly Path** | 羊毛路径 | A bio-mimetic path finding algorithm based on Slime Mold behavior for organic path generation. <br> 基于粘菌行为的仿生寻路算法，用于生成有机路径。 |
| **Minimal Surface** | 极小曲面 | A surface that locally minimizes its area, equivalent to having zero mean curvature. <br> 局部面积最小的曲面，等价于平均曲率为零。 |
| **Space Syntax** | 空间句法 | A set of theories and techniques for the analysis of spatial configurations. <br> 用于分析空间配置的一套理论和技术。 |
| **Terrain** | 地形 | The vertical and horizontal dimensions of land surface. <br> 地表的垂直和水平维度。 |
| **Class-A Surface** | A级曲面 | High-quality surfaces with curvature continuity, used in automotive and product design. <br> 具有曲率连续性的高质量曲面，常用于汽车和产品设计。 |
| **Manifold** | 流形 | A topological space that locally resembles Euclidean space; in mesh terms, "watertight" with no self-intersections. <br> 局部类似于欧几里得空间的拓扑空间；在网格术语中指“水密”且无自相交。 |
| **Graph Theory** | 图论 | The study of graphs, which are mathematical structures used to model pairwise relations between objects. <br> 研究图的数学分支，用于建模对象之间的成对关系。 |
